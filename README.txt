This README file is to indicate which file is which:

"DataScienceProjectDatast.csv" - CSV file with the database used created by me.

"DataScienceProject.Rproj" - R Project file that contains other parts of the project.

"project.R" - R file with all the code used in the analysis.

"Data Science Project Presentation.Rmd" - R Markdown file (not executable) with the code doubts mentioned in the mails.

"Data_Sci_only_images.Rmd" - R Markdown file with the presentation with images instead of the proper graphs.

"DataSci_only_images.html" - HTML file that is the proper presentation.

"*.jpg" - JPG files are all images used in the presentation.
